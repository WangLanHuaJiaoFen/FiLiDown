# FiLiDown

FiLiDown是一款基于BBDown的用flutter写的material风格的windows平台的哔哩哔哩视频下载软件。
目前功能：

- 提供BBDown大部分功能UI界面
- 一体化登录
- material风格界面
- 获取个人哔哩哔哩信息并展示
- 更改主题颜色
- 设置背景图片
- 部分参数记忆功能

未来展望：

- ......

## 使用方法

直接下载[Release](https://github.com/WangLanHuaJiaoFen/FiLiDown/releases)中的压缩包，里面已经集成了BBDown、MP4BOX、ffmpeg、aria2c，然后点开exe文件，在下载页面输入视频号或者url即可。
